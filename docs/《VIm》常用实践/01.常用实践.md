### [vim多行注释和取消注释](https://www.cnblogs.com/wenxingxu/p/9528161.html)

```vim
插入注释
1. 进入命令行模式，按ctrl + v进入 visual block模式，然后按j, 或者k选中多行，把需要注释的行标记起来

2. 按大写字母I，再插入注释符，例如//

3. 按esc键就会全部注释了


前三步与加注释相同。

（1）首先按键盘上的ESC进入命令行模式

（2）再按Ctrl+V进入VISUAL BLOCK模式

（3）利用上下左右键调整需要去注释多少行以及行的宽度

（4）再按d即可完成去注释
```
